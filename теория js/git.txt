git init - создание проэкта (создание папки гита в корневой проэкта)
git config --global(или local) user.name "Имя пользователя" - создание имени пользователя в настройках гита
git config --global(или local) user.email test@tets.ru - имейл пользователя

после файлы будут висеть не загружены в репозиторий, это можно проверить с помощью команды ниже
git status - проверка статуса файлов (если красные тогда не загружены, зеленые тогда загружены)
git add -A   - добавление файлов в репозиторий (А значит ALL)
git commit -a -m"сообщение или описание"  - создание точки контрольной версии нашего проэкта
git restore js\main.js - отменить изменения и не вносить их
git log - посмотреть комиты (точки контройльных версий)
gigit - подключил удаленый репозиторий к моему локальному
git push -u origin master - добавить файлы из локального на гитхаб (удаленный репозиторий)
git push  - закинуть файлы на гит (с локального на удаленный);
cd.. - переход в терминале в папку назад
git clone https://github.com/fashevskij/project.git project2 - делаю клон с репозитория(удаленного) в новую папку(или новый комп) с именем папки project2
git pull - загрузить файлы с удаленного репозитория на наш компьютер.
